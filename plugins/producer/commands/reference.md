---
description: Build a reference playlist on Spotify for a vibe, mix target, or track in progress
argument-hint: "<vibe, genre, or 'like this track'>"
---

Invoke the `reference-curator` skill. Pull 8–12 references via Spotify (`search` + `fetch_tracks`), tag each with BPM, key, integrated LUFS, and *the one thing* to listen for. Create the playlist on Spotify and optionally save the list to Google Drive.
